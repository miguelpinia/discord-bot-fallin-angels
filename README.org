Bot de discord para ver el estado del servidor de Fallin-Angels de Urban Terror.

* Comandos:

!map_en :: Obtiene la información del servidor en inglés.
!map_es :: Obtiene la información del servidor en español.
!next_map :: Obtiene la información del mapa siguiente.
!hello :: Envía un saludo al usuario.
